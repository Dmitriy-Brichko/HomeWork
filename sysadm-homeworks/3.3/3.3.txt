8. set -e параметр приведет к немедленному выходу сценария bash при сбое команды.
   set -u приводит к тому, что оболочка bash обрабатывает неустановленные переменные как ошибку и немедленно завершает работу.
   set -x заставляет bash печатать каждую команду перед ее выполнением.
   set -o pipefail прекращает выполнение скрипта, даже если одна из частей пайпа завершилась ошибкой.
   При использовании set -euxo pipefail скрипты bash становятся безопаснее.
